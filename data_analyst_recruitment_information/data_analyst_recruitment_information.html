4. 薪资数据(salary列)整理

该项目目的之一就是为了分析招聘薪资的下限和上限，但通过观察数据发现数据内容有小写的8k，也有大写的6K，甚至还有9K以上这种写法（这种情况我们默认上下限相同）。

我们需要用到pandas的apply函数，它可以对DataFrame中的一行或一列数据进行操作，允许使用自定义函数。

In [6]:
def cut_word(word, method):
    # 查找`-`处理 `7k-8k`此类数据
    position = word.find('-')
    length = len(word)
    
    # 处理 `7k-8k`此类数据
    if position != -1:
        bottomSalary = word[:position-1]
        topSalary = word[position+1:length-1]
        
    # 处理 `9k/9K`以上此类数据
    else: 
        bottomSalary = word[:word.upper().find('K')]
        topSalary = bottomSalary
        
    if method == 'bottom':
        return bottomSalary
    else:
        return topSalary

def topN(df, n = 5):
    '''
    将传入的数据计数，从大到小排序，返回前n的数据
    '''
    counts = df.value_counts()
    return counts.sort_values(ascending = False)[:n]
